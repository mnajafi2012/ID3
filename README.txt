
The following is the manual to run the decision tree _ID3 algorithm.
Date: 02/22/2017
Author: Mary Najafi

---
Using command line arguments:
Open a terminal and:

Examples:
1. java Main iris
2. java Main iris prune .8
3. java Main iris noise
4. java Main iris noise prune .9
5. java Main tennis
6. java Main bool
7. java Main enjoy

Using IDE (e.g. Eclipse):
Modify the Run Configuration setting of the project and under Arguments tab run:

Examples:
1. iris
2. tennis
3. bool
4. enjoy
5. iris prune .7
6. iris noise
7. iris noise prune .85
